Careers in IT


Database Manager


A database manager is in charge of creating and maintaining the systems used by an organisation to store and organise data for businesses. They guarantee the security of the data stored by putting in place a number of security programmes. The database manager is also in charge of overseeing the daily operations of the database teams and determining the data storage requirements of businesses.


Web Designer


Websites are designed, built, and improved by web designers using their innovative and software engineering/programming skills. They are able to create websites that adhere to design guidelines and standards while also being simple to understand, navigate, and use. The design and layout of a website or web page are the work of a web designer. It also refers to creating a new website or making updates to an existing one. They are occasionally referred to as front-end developers.
